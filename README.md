# Project Description
The Scholarship Eligibility Checker is a C++ application developed to assist educational institutions in determining whether students qualify for a scholarship. The program collects student information, validates the input, checks whether scholarship eligibility based on predefined rules, displays results, and saves the final reports into a text file for reference.

# Problem Statement
Many schools, colleges, and universities manually assess students for scholarship rewards. This process can be time-consuming, inconsistent, and prone to errors. There is a need for a automated system that evaluates scholarship eligibility quickly, accurately and stores results for documentation.

#Aim
To develop a C++ Scholarship Eligibility Checker that automatically determines whether students qualify for scholarship based on GPA, attendance percentage, and conduct status.


#Objectives

- To collect student details.
- To validate GPA and attendance percentage.
- To determine scholarship eligibility using conditional statement.
- To process multiple student records.
- To display eligibility results.
- To save student records and the summary report into text file.

# Scholarship Eligibility Rules.
A student is eligible for a scholarship if all the following  conditions are satisfied:
 - GPA is 3.5 or above.
 - Attendance percentage is 80% or above.
 - Conduct status is Good.
If any of these conditions are not met, the student is declined Not Eliigible for a scholarship.
